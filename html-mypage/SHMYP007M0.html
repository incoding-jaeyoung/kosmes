온라인 증명 시스템 바로가기 발급증명서 확인 제증명발급 매뉴얼

사용자 권장 환경(항목 및 권장사양)

사용자PC
  • CPU : Dual Core 이상
  • 메모리 : 1G 이상
  • OS : Win 7 이상
  • Web Browser : 구글 크롬(Chrome)

    ※ 사용 전에 PDF 뷰어가 설치되어야 함

사용자프린터
  • HP,삼성,청호등(잉크젯,레이져,복합기) : 300DPI ~ 600DPI이상
