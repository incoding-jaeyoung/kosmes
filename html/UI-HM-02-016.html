진행상태

  • 임시저장 : 신청서 작성을 완료하지 않고 임시로 저장한 상태입니다.
  • 신청서제출 : 신청서 제출이 완료된 상태입니다.
  • 심사중 : 신청서를 접수하여 성과창출 여부를 확인 중입니다.(성과확인 약1~2개월 소요)
  • 환급완료 : 성과창출이 확인되어 환급금 지급이 완료된 상태입니다.
  • 반려 : 귀사의 요청으로 신청이 취소된 상태입니다. (신청서 재작성 가능)
  • 환급불가 : 성과창출 기준 미충족, 휴/폐업, 기타 사유로 인해 환급이 불가합니다.
